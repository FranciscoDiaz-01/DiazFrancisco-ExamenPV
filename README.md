Página para turnos médicos 
Características principales 

 Para pacientes
- Registro e inicio de sesión con datos personales.
- Navegar profesionales por especialidad y ver su ficha.
- Reservar turnos disponibles (solo horarios de la mañana para un día dado).
- Consultar el historial de citas agendadas y ver detalles.
- Descarga de comprobante de cita (funcionalidad en planificación/desarrollo).

Para médicos
- Registro e inicio de sesión como profesional.
- Crear y gestionar horarios (agregar o eliminar franjas).
- Visualizar la lista de turnos reservados y ver información básica del paciente.
- Cancelar citas cuando sea necesario y dejar anotaciones simples.
